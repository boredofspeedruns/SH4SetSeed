# SH4 Set Seed (Silent Hill 4 RNG & HUD Mod)

## Features

- Adds an overlay with an onscreen timer for verification purposes.
- Controls two RNG systems via custom seeds:
  - **Hospital L1/R1 elevator doors**
  - **One Truth boss**
- Reads settings from a simple `sh4setseed.ini` file.
- Lets you **cycle One Truth index live** with a hotkey (F7).
- Works with the GOG version with and without DxWnd.

> Not affiliated with or endorsed by Konami. Use at your own risk.

## How to Install / How to Use

Easy.
1. Copy the d3d8.dll, og_d3d8.dll, sh4setseed.dll and sh4setseed.ini to your SILENT HILL 4 folder.
2. Ensure that sh4setseed.ini has Enabled set to 1.
3. Enjoy!
